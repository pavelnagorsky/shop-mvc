An online shop website.
_______________________________

Static rendering by MVC pattern.

There are: 
  - express & mongoose
  - CRUD operations above the products,
  orders, cart and more...
  - User authentication with sessions
  and crypted by cryptjs passwords
  - CRSF token protection
  - Multer for storing files
  - NodeMailer for resetting passwords 
  and notifications
  - Creating a .pdf invoice for each 
  order
  - Stripe in testing mode as a payment gateway
